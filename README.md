 Retail-Sales-Analysis

This project analyzes retail sales data using SQL and Power BI to extract key business insights.

 Project Overview
- Used the Superstore dataset to analyze sales, profits, customer behavior, and regional trends.
- Performed data aggregation and querying using SQL.
- Created interactive dashboards and visualizations in Power BI.

 Tools Used

- SQL (MySQL / any other you used)
- Power BI Desktop

Key Analyses and Insights

- Total sales, profits, and discounts overview.
- Sales and profit breakdown by region, segment, and category.
- Top 10 products and customers by sales.
- Monthly sales trends and growth rates.
- Profit margins and impact of discounts on profitability.

 Files Included- SQL scripts for data queries.
- Power BI dashboard file (.pbix).
- Project summary document.
